《现代汉语常用词表(草案)》由厦门大学国家语言资源监测与研究教育教材中心于2008年出版，2015年开始对草案进行修订，于2021年出版《现代汉语常用词表》(第2版)，共收词56790个。包括单音节词3149个,双音
节词40924个,三音节词6583个,四音节词5966个,五音节以及五音节以上词语 168个。

经本人使用ORC工具提取，再经人工校对，达到与原书主体内容99.9%的相似度，字词排序与原书保持一致。

感觉有不少词并不常见，有些是方言用词，原书还有标错拼音，就本人发现的有接近二十处，如“杭育（hángyo）”就标成了hángyāo，“核试验”标成了shí，可能是从草案中的“核实验”改为“核试验”时拼音未改过来。
