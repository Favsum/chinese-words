由国家语委成立的《现代汉语通用词表》研制课题组，于2008年出版《现代汉语常用词表(草案)》，2015年开始对草案进行修订，于2021年出版《现代汉语常用词表》(第2版)，共收词56790个。包括单音节词3149个,双音
节词40924个,三音节词6583个,四音节词5966个,五音节以及五音节以上词语168个。

使用ORC工具提取，再经人工校对，达到与原书主体内容99.9%的相似度，词表排序与原书一致。

词表中有不少词并不常见，有些是方言用词，原书有近二十处拼音标错，如“杭育（hángyo）”标成了hángyāo，“核试验”的试标成了shí，疑似从草案中的“核实验”改为“核试验”时拼音未一起改过来。
